2.预备知识

2.1 主要模块

  1. Core 核心模块,包括基础数据结构,绘图函数,数组操作函数.

  2. Imgproc: 图像处理模块, 图像滤波,几何变换,直方图等等

  3. Highgui 顶层GUI模块

  4. Video : 视频模块包括运动分析和目标跟踪

  5. 其他模块:包括Calib3d Features2d等

2.2 数字图像本质: 矩阵

  • OpenCV中的Mat类

  • Java中调用OpenCV函数需要加上模块名

  • HighGui.imshow(“lena”,src);

  • IMREAD_COLOR：加载彩色图像，任何图像的透明度都会被忽略，它是默认标志

  • IMREAD_GRAYSCALE：以灰度模式加载图像

  • IMREAD_UNCHANGED：加载图像，包括 alpha 通道

  • 读取图像: Mat src = Imgcodedcs.imread(“lena.jpg”);

  • 输出图像 Imgcodecs.imwrite(“lena.jpg”,mat);

  • 创建每个点只有一个通道的简单矩阵可以使用:zeros (全为0),eys(自反,其他为0),ones(全为1)

  • 一个图像的通道数是N，就表明每个像素点处有N个数，一个a×b的N通道图像，其图像矩阵实际上是b行N×a列的数字矩阵。

  1. 通道的是灰度图。

  2. 通道的图像是RGB555和RGB565。2通道图在程序处理中会用到，如傅里叶变换，可能会用到，一个通道为实数，一个通道为虚数，主要是编程方便。RGB555是16位的，2个字节，5+6+5，第一字节的前5位是R，后三位+第二字节是G，第二字节后5位是B，可见对原图像进行压缩了。

  3. 通道的是彩色图像，比如RGB图像。

  4. 通道的图像是RGBA，是RGB加上一个A通道，也叫alpha通道，表示透明度。PNG图像是一种典型的4通道图像。alpha通道可以赋值0到1，或者0到255，表示透明到不透明。

  • CV_8UC1 : 8位 Unsigned(0-255) channel = 1 灰度图

  • CV_8UC3 : 8位 Unsigned(0-255) channel = 3 彩色

  • CV_64FC3 : 64位浮点数 channel = 3 彩色

4.2 图像平滑

  • 椒盐噪声: 数字图像的一个常见噪声，所谓椒盐。 椒就是黑，盐就是白，椒盐噪声就是在图像上随机出现黑色白色的像素。

示意图

  • 高斯噪声:概率密度函数服从高斯分布

  • 卷积核:权重矩阵

  • 均值滤波 blur(Mat src, Mat dst, Size ksize) //算数平均值

  • ksize 表示卷积核大小,越大越模糊

  • 高斯滤波 GaussianBlur(Mat src, Mat dst, Size ksize, double sigmaX, double sigmaY, int borderType) //实际上让低的向中心权重靠近

  • 中值滤波 medianBlur(Mat src, Mat dst, int ksize) //顾名思义就是中值 去除椒盐噪音

5. 腐蚀、膨胀、开闭操作、礼帽与黑帽形态学操作

  1. 概念：

  • 腐蚀和膨胀，最基础的形态学操作

  • 膨胀：求局部最大值的操作，使图像的高亮区域逐渐增长

  • 腐蚀：求局部最小值操作，使图像的高亮区域逐渐减小

  1. 具体操作

  • 腐蚀: 结构元素扫描目标图像，每找到一个完全相同的子结构，就将目标图像原点位置1 ，否则为0

  • 膨胀： 作与运算之后如果都为0则为0 ，否则为1.

  1. 相关函数：

  2. 膨胀函数dilate 有3个原型方法，前两种较常用；

  • dilate(Mat src, Mat dst, Mat kernel)；

  • dilate(Mat src, Mat dst, Mat kernel, Point anchor, int iterations)；

  • dilate(Mat src, Mat dst, Mat kernel, Point anchor, int iterations, int borderType, Scalar borderValue)；

  1. 参数说明：

  • kernel : Mat 膨胀操作的核 ，null时表示以当前像素为中心 3x3 的核；

  • anchor : 瞄点，默认值为(-1,-1)，代表这个核的中心位置；

  • iterations : Integer 迭代次数，默认为1；

  • borderType : Integer 推断图像外部像素的某种边界模式，默认值为BORDER_DEFAULT；

  • borderValue : Scalar 当 borderType值为常数时有默认值，一般不用管；

  1. 腐蚀函数erode也 有3个原型方法，参数同dilate函数，说明略；

  2. 形态学操作：Imgproc.morphologyEx(Mat src, Mat dst, int op, Mat kernel)；

  • Imgproc.MORPH_OPEN：开运算，先腐蚀后膨胀

  • Imgproc.MORPH_CLOSE：闭运算：先膨胀后腐蚀

  • Imgproc.MORPH_GRADIENT：形态学梯度，图像膨胀与腐蚀的差，看上去就像轮廓

  • Imgproc.MORPH_TOPHAT：进行开运算之后得到的图像与原始图像的差；

  • Imgproc.MORPH_BLACKHAT：进行闭运算之后得到的图像与原始图像的差；
