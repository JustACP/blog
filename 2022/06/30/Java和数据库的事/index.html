初识 JDBC

JDBC(Java Data Base Connectivity) Java数据库连接,Java 和广泛数据库之间独立于书u艰苦的连接标准的Java API。
从根本上讲：JDBC是一种规范，它提供的接口，是一套完整的、允许便携式访问底层数据库的。可以用Java来写不同类型的可执行文件：Java应用程序、Java Applets、Java Servelet、Jsp等。
就是用Java 操作数据库。

JDK自带了 java.sql包，里面定义了大量接口。不同数据库都可以通过这个实现此接口。数据库驱动

管理事务

JDBC默认的事务处理行为是自动提交，所以前面我们执行一个SQL语句就会被直接提交（相当于没有启动事务），所以JDBC需要进行事务管理时，首先要通过Connection对象调用setAutoCommit(false) 方法, 将SQL语句的提交（commit）由驱动程序转交给应用程序负责。

一旦关闭自动提交，那么现在执行所有的操作如果在最后不进行 commit()来提交事务的话，那么所有的操作都会丢失，只有提交之后，所有的操作才会被保存！也可以使用 rollback()来手动回滚之前的全部操作！
